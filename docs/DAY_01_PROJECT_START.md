# Day 1: Project Start

## Goal

Start the capstone project with a clear idea, project structure, technology stack, and submission evidence plan.

## Selected Project Idea

SecureBlog - Cloud-Based Blog and News Portal

## Why This Project Is Suitable

This project is simple enough to finish on time, but strong enough to demonstrate all required areas:

- Application development
- User and admin login
- Blog/news post CRUD operations
- AWS cloud infrastructure
- Linux administration
- Networking and domain setup
- Cyber security
- DevOps automation
- Monitoring and logging

## Main User Roles

### User

- Register account
- Login
- Create blog/news post
- View own posts
- Edit own posts
- Delete own draft posts
- View published posts

### Admin

- Login
- View all users
- View all posts
- Approve or reject posts
- Edit or delete any post
- View registered users

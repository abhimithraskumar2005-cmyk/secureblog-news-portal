# Day 6: Frontend Pages

## Goal

Create a usable frontend for the SecureBlog Blog/News Portal.

## Completed Work

- Public published posts section
- Register form
- Login form
- User dashboard
- Create post form
- Edit post support
- Delete post support
- Admin dashboard
- Admin publish/reject controls
- Frontend API integration with backend

## Local Run Commands

Terminal 1:

```bash
cd backend
npm run dev
```

Terminal 2:

```bash
cd frontend
npm start
```

Frontend URL:

```text
http://localhost:5173
```

# SecureBlog - Cloud-Based Blog and News Portal

Cloud-based DevSecOps capstone project.

## Project Objective

The objective of this project is to design, develop, secure, deploy, automate, and monitor a cloud-based Blog/News Portal that demonstrates core DevSecOps skills. The application allows users to create blog/news posts and allows administrators to review, approve, reject, and manage posts through role-based access control.

## Planned Features

- User registration and login
- Role-based access: User and Admin
- Blog/news post CRUD: create, read, update, delete
- JWT authentication
- Password hashing using bcrypt
- API rate limiting
- Security headers
- HTTPS deployment
- AWS cloud hosting
- CI/CD pipeline
- Monitoring and logging

## Planned Technology Stack

- Backend: Node.js, Express.js
- Database: PostgreSQL or MySQL
- Authentication: JWT
- Password Security: bcrypt
- Reverse Proxy: Nginx
- Containerization: Docker
- Cloud: AWS EC2, VPC, ALB, RDS, Route 53, CloudWatch
- CI/CD: GitHub Actions
